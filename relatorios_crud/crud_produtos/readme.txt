# 🛠️ ABAP - CRUD Simples com Produtos

Este projeto demonstra um CRUD completo em ABAP usando uma tabela personalizada `ZPRODUTOS`.

## 📦 Funcionalidades

- Criar produto (com ID gerado automaticamente)
- Consultar produto por ID
- Atualizar produto
- Excluir produto

## 🧱 Conceitos aplicados

- Operações `INSERT`, `SELECT`, `UPDATE`, `DELETE`
- Validação de campos obrigatórios
- Uso de `SY-DATUM` para data de criação
- Separação por `FORM routines`
- Mensagens de retorno para o usuário

## ✅ Objetivo

Praticar e demonstrar o domínio de operações CRUD em ABAP, com boas práticas de validação e estruturação do código.
