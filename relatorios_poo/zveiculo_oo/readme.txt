# 🚗 ABAP - Exemplo de POO com Veículos

Projeto ABAP simples que demonstra os conceitos principais de Programação Orientada a Objetos usando veículos como exemplo.

## 🧱 Funcionalidades

- Classe abstrata (`ZCL_VEICULO`)
- Herança (`ZCL_CARRO`, `ZCL_MOTO`)
- Encapsulamento (atributos privados)
- Polimorfismo (uso de referência genérica)
- Construtores personalizados

## 🧪 Execução

Instancia objetos `CARRO` e `MOTO` e exibe suas informações com método polimórfico.

## ✅ Objetivo

Demonstrar domínio de POO em ABAP para fins didáticos e portfólio.
